# AuraCast

AuraCast is a Solana-powered platform that lets creators monetize their voice using ElevenLabs AI voice cloning and Solana Actions (Blinks).

## Getting Started

1. Fill in `.env.local` with your API keys.
2. `npm run dev`

## Stack

- Next.js 14 App Router
- Solana Web3.js + Actions
- ElevenLabs Voice API
- Supabase
- Anthropic Claude (moderation)
